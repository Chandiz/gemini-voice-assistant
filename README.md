# Gemini Voice Assistant (Raspberry Pi / PC)

A simple voice-based AI assistant using Google's Gemini API.
Originally built on Raspberry Pi 3B+, but runnable on any PC with Python.

## Features
- Voice input via microphone
- AI responses using Gemini API
- Continuous conversation loop
- Lightweight and simple setup

## Requirements
- Python 3.9+
- Microphone
- Google Gemini API key

## Installation
```bash
pip install -r requirements.txt
